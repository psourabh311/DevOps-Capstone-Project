This repository contains the DevOps Capstone Project developed as part of the Coursera DevOps Capstone Project course.

The objective of this project is to design, build, and deploy a cloud-native microservices application using industry-standard DevOps practices. The project follows Agile methodology with Scrum, incorporates CI/CD pipelines, and applies automation across build, test, and deployment stages.

Key practices demonstrated include:

Agile project planning using GitHub Kanban boards and user stories

Continuous Integration using GitHub Actions

Continuous Delivery with automated pipelines

Containerization using Docker

Orchestration using Kubernetes

Infrastructure and application deployment using cloud-native principles

Monitoring, testing, and quality assurance best practices

This capstone simulates a real-world DevOps environment, covering the full lifecycle from planning and backlog creation to production deployment.
